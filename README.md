loan-calculator
===============

住宅ローン計算機

### Install

```
$ go get -u github.com/li-go/loan-calculator
```

### Usage

```
$ loan-calculator
Usage of loan-calculator:
  -ny int
        借入年数 (default 35)
  -p int
        借入金額 (default 50000000)
  -r float
        利息(%) (default 0.525)
exit status 2
```
